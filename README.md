# Ravindra Jadeja Tribute Page

A simple tribute page dedicated to the Indian cricketer Ravindra Jadeja, showcasing his personal and professional information.

## Description

This project is a static web page built with HTML and CSS that honors Ravindra Jadeja, also known as "Jaddu" or "Thalapathy". The page includes his personal details, international cricket statistics, and domestic team information.

## Features

- Clean and responsive design
- Personal information section
- International cricket career details
- Domestic team information
- Styled with CSS for better presentation

## Technologies Used

- HTML5
- CSS3

## How to View

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. Ensure the image file `Ravindra Jadeja.jpg` is in the same directory as the HTML file
